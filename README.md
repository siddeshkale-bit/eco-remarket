# Eco Remarket 📦

Eco Remarket is a Flask-based e-commerce platform designed for resellers and creators to trade electronics and toys.

## 🚀 Features
* **Guest Browsing:** View products without an account.
* **Smart Search:** Filter by name, category, or sub-category.
* **Role-Based Access:** * **Creators:** Upload and manage products.
    * **Resellers:** Shopping cart and order tracking.
    * **Admin:** Full system management.

## 🛠️ Tech Stack
* **Backend:** Python, Flask, SQLAlchemy
* **Frontend:** HTML5, Bootstrap 5, Animate.css

## 🔧 Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
